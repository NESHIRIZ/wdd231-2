Africhique Website Project – Site Plan

1. Site Name

Africhique Designs

This name was selected because it blends “Africa” and “chic,” clearly representing a modern African clothing brand that focuses on stylish, high-quality African print designs.

2. Site Purpose

The purpose of the Africhique website is to showcase African-printed clothing, provide product information, allow users to browse new designs, place custom clothing orders, and contact the business for tailoring or inquiries. It will act as a digital storefront for both Ghanaian and international customers.

3. Scenarios

  • Where can I browse Africhique’s latest outfits, styles, and prices?
  • How do I place a custom order for African wear tailored to my size and design?
  • Does Africhique offer delivery or international shipping?

4. Color Scheme

Primary Color – #2A7F62 (Green): Used for headings, borders, navigation accents, and section titles.

Accent Color – #F2C94C (Gold): Used for highlights, buttons, call-to-action areas, and emphasis text.

5. Typography

  • Poppins – Used for all headings and body text because it is clean, modern, and highly readable.
  • Arial – Backup font for body content to ensure readability if Poppins fails.

This document is currently using the selected typography.
